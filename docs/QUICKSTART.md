# Quick Start Guide

## Installation

1. **Install scrcpy:**
   ```bash
   brew install scrcpy
   ```

2. **Make script executable:**
   ```bash
   chmod +x android-mirror.sh
   ```

3. **Run the script:**
   ```bash
   ./android-mirror.sh
   ```

## Quick Test

```bash
# Check if scrcpy is installed
which scrcpy

# Check connected devices
adb devices

# Run the script
./android-mirror.sh
```
